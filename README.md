# Mini Airbnb Project

## Project Overview
This project is a mini Airbnb clone including a login system, property listings, a map view, and a new market analysis dashboard.

## Key Features
- **Login System**: Secure access (User: `admin`, Pass: `1234`).
- **Property Listings**: View available resorts and hotels.
- **Interactive Map**: Visualize property locations.
- **Analytics Dashboard**: 
  - Compare market prices.
  - Calculate estimated profits based on occupancy.
  - View seasonal trends.

## File Structure
- `index.html`: Login page.
- `home.html`: Main dashboard with listings.
- `map.html`: Google Maps integration.
- `analysis.html`: **NEW** Analytics and Profit Calculator.
- `style.css`: Global styling and responsive design.
- `script.js`: Logic for login and interactions.

## How to Run
1. Open `index.html` in your browser.
2. Login to access the full site.
